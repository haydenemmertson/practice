Coreform Cubit Installation
============================

Coreform Cubit is a software designed to generate a mesh for any geometry. For the purpose of this software it will be used ot generate a simple tetrahedral mesh. CoreformCubit can be downloaded here_.

A replacement software must have the capabilites to generate a tetrahedral mesh, define nodesets, and export files in an 'inp' format.

.. _here: https://coreform.com/downloads/
