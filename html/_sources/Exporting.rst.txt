Exporting
==========

Once a proper mesh has been generated and the side sets defined, the mesh must be exported in a format that is easy to for the algorithm to read.


The GitHub repository algorithm reads in the elements and in the form of an Abacus file. To export the mesh, select the export button, name the file something recognizable, then select the export button, name the file something recognizable, then select Abacus fro the file type, as shown in the image below.
