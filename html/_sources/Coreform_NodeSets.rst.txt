1. Navigate to the Command Panel and select the 'Analysis Groups and Materials' icon.

  .. figure:: _static/NodeSet_a.png
     :alt: Image supporting the first step of the nodeset process
     :width: 400px
     :align: center

2. Then select the 'Nodesets' icon as shown in the figure below.

  .. figure:: _static/NodeSet_b.png
     :alt: Image supporting the second step of the nodeset process
     :width: 400px
     :align: center

3. Next select the 'Create Nodesets' icon as shown in the figrue below.

  .. figure:: _static/NodeSet_c.png
     :alt: Image supporting the third step of the nodeset process
     :width: 400px
     :align: center

4. In the 'Nodeset Name' location, place the name the side set will be known as. This name of the nodeset will be used later when generating the sweep.

  .. figure:: _static/NodeSet_d.png
     :alt: Image supporting the fourth step of hte nodeset process
     :width: 400px
     :align: center

5. Next change the selection type to 'Surface'.

  .. figure:: _static/NodeSet_e.png
     :alt: Image supprting the fifth step of the nodeset process
     :width: 400px
     :align: center

6. Click the textbox for 'ID(s)' selection box.

     :alt: Image supporting the sixth step of the nodeset process
     :width: 400px
     :align: center
